# NHẬN DIỆN KHUÔN MẶT (FACE RECOGNITION)
## Giới thiệu
Đây là project nhận diện khuôn mặt được xây dựng nhằm nghiên cứu và ứng dụng các kỹ thuật xử lý ảnh và thị giác máy tính. Chương trình cho phép phát hiện và nhận diện khuôn mặt người dùng thông qua webcam theo thời gian thực.
## Mục tiêu
- Tìm hiểu về xử lý ảnh và nhận diện khuôn mặt
- Ứng dụng thư viện OpenCV trong Python
- Xây dựng chương trình nhận diện khuôn mặt cơ bản
- Làm quen với quy trình quản lý mã nguồn bằng Git và GitHub
## Tính năng
- Phát hiện khuôn mặt theo thời gian thực
- Nhận diện khuôn mặt đã được huấn luyện
- Hiển thị tên người dùng trên khung hình
- Hoạt động với webcam máy tính
## Công nghệ sử dụng
- Ngôn ngữ lập trình: Python 3
- Thư viện: OpenCV, NumPy
- Công cụ: Visual Studio Code, Git, GitHub
## Kết quả
Chương trình có khả năng phát hiện và nhận diện khuôn mặt thông qua webcam với độ chính xác tương đối trong điều kiện ánh sáng phù hợp.
## Ghi chú
- Webcam phải hoạt động để chương trình chạy đúng
- Không commit các file build, file thực thi (.exe)
- Project mang tính học tập và nghiên cứu
